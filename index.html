Spatial Model Checking for Medical Image Analysis

VoxLogicA is a spatial model checker that treats medical images as spatial structures and evaluates logical specifications over them. Image-analysis procedures are written in ImgQL, a language whose operators describe intensity, distance, reachability, connectedness, and region growth. The specification itself is the inspectable, revisable artifact — not an opaque pipeline.

The semantic foundation is closure spaces; spatial logics over these spaces provide the formal link between topological concepts and image-level reasoning. The implementation combines model-checking algorithms with image-processing libraries and parallel execution for 3D medical volumes.

Case studies focus on brain MRI: glioblastoma segmentation (BraTS 2017/2019/2020) and tissue segmentation in healthy brains (BrainWeb). Recent work extends the purely symbolic approach with hybrid symbolic–subsymbolic procedures, where learned components (e.g. nnU-Net predictions) interact with logical specifications, and with browser-based interfaces for visualization and collaborative analysis.

A short ImgQL example for lesion segmentation:

let flair = intensity(flair)
let hi    = flair >. 0.95
let vi    = flair >. 0.86
let hyper = flt(5.0, hi)
let very  = flt(2.0, vi)
let lesion = touch(hyper, very)
